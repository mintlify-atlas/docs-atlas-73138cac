The Metabase Embedding SDK is available as an npm package that you can install using your preferred package manager.

## Prerequisites

Before installing the SDK, make sure you have:

- Node.js 16 or higher
- React 18 or higher
- A Metabase instance (Pro or Enterprise plan)

## Installing the SDK

<CodeGroup>

```bash npm
npm install @metabase/embedding-sdk-react
```

```bash yarn
yarn add @metabase/embedding-sdk-react
```

```bash pnpm
pnpm add @metabase/embedding-sdk-react
```

</CodeGroup>

## Setting up Metabase

If you don't have a Metabase instance yet, you can:

### Option 1: Start a free trial

Start a free trial of [Metabase Pro](https://www.metabase.com/pricing/) or Enterprise.

### Option 2: Run locally with Docker

Run Metabase Enterprise locally using Docker:

```bash
docker run -d -p 3000:3000 --name metabase metabase/metabase-enterprise:latest
```

### Option 3: Download the JAR

Download and run the [Metabase Enterprise JAR](https://downloads.metabase.com/enterprise/latest/metabase.jar):

```bash
java --add-opens java.base/java.nio=ALL-UNNAMED -jar metabase.jar
```

By default, Metabase will run at `http://localhost:3000`.

<Note>
For more installation options, see the [Metabase installation docs](https://www.metabase.com/docs/latest/installation-and-operation/installing-metabase).
</Note>
